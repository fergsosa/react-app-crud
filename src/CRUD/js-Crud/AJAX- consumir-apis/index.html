AJAX (JavaScript Asíncrono y XML)

🔻

Es un conjunto de técnicas de desarrollo web que permiten crear aplicaciones web más dinámicas y responsivas. En lugar de recargar toda la página cada vez que el usuario realiza una acción, AJAX permite actualizar partes específicas de una página de forma asíncrona, es decir, en segundo plano, sin que el usuario note la demora.

¿Qué es AJAX?

🔻

AJAX es un acrónimo que significa Asynchronous JavaScript and XML. En español, esto se traduce como JavaScript Asíncrono y XML. Aunque su nombre menciona XML, actualmente se utiliza más comúnmente con JSON para intercambiar datos. En esencia, AJAX es un conjunto de técnicas de desarrollo web que permiten crear aplicaciones web más dinámicas y responsivas. Estas aplicaciones pueden actualizar partes específicas de una página web sin necesidad de recargar toda la página, lo que resulta en una experiencia de usuario mucho más fluida.

¿Cómo funciona AJAX?

🔻
  1. El usuario interactúa con la página: Por ejemplo, hace clic en un botón, escribe en un campo de búsqueda o selecciona una opción de un menú desplegable.
  2. JavaScript detecta la acción: El código JavaScript que se encuentra en la página web detecta esta interacción del usuario.
  3. Se envía una solicitud al servidor: A través de JavaScript, se envía una solicitud al servidor web en segundo plano. Esta solicitud puede ser para obtener datos, actualizar información o realizar alguna otra acción.
  4. El servidor procesa la solicitud: El servidor recibe la solicitud, la procesa y envía una respuesta. Esta respuesta puede ser en formato XML o JSON, pero como mencioné antes, JSON es el formato más utilizado actualmente.
  5. JavaScript recibe la respuesta: JavaScript en la página web recibe la respuesta del servidor.
  6. Se actualiza la página: Utilizando el DOM (Document Object Model), JavaScript actualiza la parte correspondiente de la página web con los datos recibidos del servidor.
Ejemplo práctico:

🔻
Imagina que estás en una tienda en línea y quieres buscar un producto. Al escribir en el campo de búsqueda, AJAX entra en acción:
  • Escribes unas palabras clave.
  • JavaScript detecta tu acción y envía una solicitud al servidor con las palabras clave que escribiste.
  • El servidor busca en su base de datos los productos que coinciden con esas palabras clave y envía los resultados en formato JSON.
  • JavaScript recibe los resultados y actualiza la sección de resultados de búsqueda en la página, mostrando los productos encontrados sin necesidad de recargar toda la página.
Ventajas de AJAX:

🔻
  • Mayor interactividad: Las aplicaciones web se vuelven más dinámicas y responsivas.
  • Mejor experiencia de usuario: Las páginas cargan más rápido y se actualizan sin necesidad de recargar toda la página.
  • Menor consumo de ancho de banda: Al cargar solo los datos necesarios, se reduce el tráfico de datos.
  • Aplicaciones web más ricas: Se pueden crear aplicaciones web con características similares a las aplicaciones de escritorio.
